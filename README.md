# ナツノヨノユメ 投票・アンケートLP

HRWC 2026 | LIVE WITH ROBOTS(2026.7.18–19 アスティとくしま)
劇『ナツノヨノユメ』終演時、Robovie-R4 が提示するQRコードの遷移先サイト。

## 内容

- 投票: 夢を見てしまった「故障」したロボットを処分するか(処分する / 処分しない)
- アンケート: 感想の自由記述+年代(任意)

## デザイン検討用パス

| パス | パターン |
|---|---|
| `/` | 選択ページ |
| `/a/` | A: 月夜 — 静謐ミニマル |
| `/b/` | B: 機体内部 — システムHUD |
| `/c/` | C: 古書 — 羽ペンとインク |

採用パターン決定後、ルート直下に本番ページを配置する。

## アーキテクチャ

- 配信: GitHub Pages(静的・CDN)— 終演直後のアクセス集中はページ側で吸収
- 投票/感想の保存: Google フォーム formResponse へJSから送信(未接続。`sendToBackend()` がスタブ)
- リアルタイム表示: スプレッドシート → Apps Script(JSON) → 結果表示ページ(未実装)
